## 확인 가능한 제한

- 운영 DB에는 현재 인수자 ETF가 0개다. 운영 데이터를 인위적으로 만들지 않았으므로 라이브 이자 지급은 실행하지 않았고 서비스 테스트와 드라이런으로 검증했다.
- 운영 시점에 적대적 M&A 대상 인수자 ETF가 없어 라이브 M&A 완료 흐름은 실행하지 않았다.
- 기존 서비스에는 WebSocket 구현이 없다. 이번 적용도 기존 polling·HTTP 구조를 유지했다.
- 기존 서비스에는 자동 시즌 종료 스케줄러가 없으며 시즌 종료는 관리자 제어 흐름을 사용한다.
- Vite 빌드는 성공했지만 약 542KB JavaScript 청크에 대한 크기 경고가 남아 있다.
- 배포 보조 스크립트의 CRLF 문제로 첫 재시작이 자동 실행되지 않아 수동으로 서비스를 시작했다. 확인된 비가동 시간은 약 29초였고 데이터 손상은 없었다. 릴리스 전환 과정의 추가 비가동은 약 15초였다.
